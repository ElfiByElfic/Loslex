Jak fungují e-maily

Aplikace sama e-maily neposílá — k tomu je potřeba propojení s e-mailovou službou. Nejjednodušší možnosti:

📅 iCalendar (doporučeno): Exportuj sledované závody jako .ics soubor a přidej do Google / Apple kalendáře. Kalendář ti sám pošle upomínku e-mailem.

⚡ Make.com / Zapier: Stáhni JSON a nastav automatizaci, která denně zkontroluje termíny a pošle e-mail.

🐍 Vlastní skript: Repozitář obsahuje scripts/scrape.py a hotový GitHub Actions workflow — stačí přidat krok s odesláním e-mailu (SendGrid, SMTP).
